# Rintik Hujan  

Sebuah halaman web lokal dengan suasana hutan hujan neon biru.  

## Fitur  
- Latar belakang gelap kebiruan dengan gradien radial  
- Animasi tetesan hujan partikel menggunakan Canvas  
- Suara hujan loop sebagai ambience  

## Cara menjalankan  
1. Unduh semua file ke folder `rintik_hujan`.  
2. Pastikan file `hujan.mp3` berada di folder yang sama.  
3. Buka file `index.html` di browser.  

## Teknologi  
HTML + CSS + JavaScript (Canvas)  

## Lisensi  
Gunakan bebas untuk keperluan pribadi atau publik. Jika kamu menggunakan suara dari sumber bebas hak cipta, periksa lisensi sumbernya.
